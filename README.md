# Random Color Generator

This is a small frontend project built with **Vite**.\
The application generates a random color and allows the user to copy its
hex code.

## Setup and Run Instructions

1.  Clone the repository

```{=html}
<!-- -->
```
    git clone <repository-url>
    cd css-build-demo

2.  Install dependencies

```{=html}
<!-- -->
```
    npm install

3.  Run the development server

```{=html}
<!-- -->
```
    npm run dev

4.  Build the project

```{=html}
<!-- -->
```
    npm run build

After building, the production files will be generated in the `dist`
folder.

## How the CSS is Transformed

The CSS for this project is written in `src/style.css` and imported into
`src/main.js`.

During the build process, **Vite collects the CSS from the project and
bundles it into a generated CSS file**. The final CSS used by the
browser is created during the build step and placed inside the
`dist/assets` directory.

The generated CSS file does not correspond one-to-one with the original
source files because it is processed and bundled by the build tool. Vite
also assigns a hashed filename to the generated files.

Source maps are enabled in the project using the following configuration
in `vite.config.js`:

``` js
export default defineConfig({
  build: {
    sourcemap: true
  },
  css: {
    devSourcemap: true
  }
});
```

## Generated CSS and Source Maps

After running the build command:

    npm run build

The generated files can be found in:

    dist/assets/

Example files generated by the build process:

    index-XXXXX.css
    index-XXXXX.js
    index-XXXXX.js.map

These files represent the final assets used by the browser.
